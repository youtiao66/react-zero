# react-zero
从零开始搭建 react 项目

## `react + typescript + webpack`
基础项目搭建

- 创建模板文件和入口文件 `index.html + App.tsx`
- 添加 `webpack.config.js + tsconfig.json` 进行项目构建
  - 使用 `ts-loader` 处理 `jsx`
  - `tsconfig.json` 需要添加 `"jsx": "react-jsx"`

## `react-router-dom`
新增路由

- `webpack-dev-server` 快速搭建本地开发服务
- `path alias` 配置路径别名
- `react-router-dom` 新增路由
